## Overview

This SOP provides a standardized methodology for reviewing network device configurations including routers, switches, firewalls, and load balancers. The procedure aligns with ISO 27001 Annex A controls (A.13.1 - Network Security Management).

## Contents

1. **Scope and Objectives**
2. **Pre-Assessment Activities**
3. **Configuration Review Checklist**
   - Router/Switch Hardening
   - Firewall Rule Analysis
   - VLAN Segmentation
   - Access Control Lists
4. **Evidence Collection**
5. **Reporting Template**
6. **Remediation Tracking**
